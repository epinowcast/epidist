# Rename the columns of a `data.frame`

Rename the columns of a `data.frame`

## Usage

``` r
.rename_columns(data, new_names, old_names)
```

## Arguments

- data:

  A `data.frame` to rename the columns of.

- new_names:

  A character vector of new column names.

- old_names:

  A character vector of old column names.
