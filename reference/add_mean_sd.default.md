# Default method for add natural scale parameters

Default method for add natural scale parameters

## Usage

``` r
# Default S3 method
add_mean_sd(data, ...)
```

## Arguments

- data:

  A dataframe of distributional parameters.

- ...:

  Additional arguments for method.
